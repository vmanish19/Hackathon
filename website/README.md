# WebSocket Spec

The data to communicate between hardware and web

## Host -> Web
- `0:0` - Red Button
- `1:0` - Yellow Button
- `2:0` - Green Button
- `3:0` - Jog Wheel Button
- `4:0` - Left wheel turn
- `5:0` - Right wheel turn
- `6:<int>` - Volume where **<int>** is the volume between 0 - 35

## Web -> Host
